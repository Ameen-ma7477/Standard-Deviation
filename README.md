# Height Data Analysis with Python

This project explores a height dataset using Python, Pandas, Seaborn, and Statistics.

## Topics Covered

- Data loading with Pandas
- Descriptive statistics
- Mean and Standard Deviation
- The 3-Sigma Rule
- Outlier Detection
- Z-Score Calculation
- Data Cleaning

## Technologies Used

- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib

## Key Learning

I learned how outliers can affect a dataset and how to identify them using statistical methods such as the 3-sigma rule and Z-scores.
